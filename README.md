# VERSUS - Football Comparison

A sleek, dark-themed football player and team comparison website inspired by Sorare's premium sports aesthetic.

## Features

### Player Comparison
- Compare 2-4 players side by side
- Smart stats including xG, xA, goals per 90, conversion rates
- Categories: Basic, Attacking, Shooting, Playmaking, Physical & Duels, Efficiency
- Highlights the best performer in each stat

### Team Comparison
- Compare 2-4 teams side by side
- Advanced metrics including xG for/against, possession, set piece analysis
- Categories: League Standing, Attack, Shooting, Possession, Defense, Play Patterns, Form
- Highlights the leading team in each stat

## Tech Stack
- Pure HTML, CSS, and JavaScript
- No frameworks or dependencies
- Responsive design

## Usage

1. Open `index.html` in your browser
2. Select "Players" or "Teams" tab
3. Click on cards to select (max 4)
4. View detailed stat comparisons below
5. Gold highlighting shows the best in each category
